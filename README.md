# agent-skills

A collection of agent skills for AI coding assistants.

## Installation

Install all skills:

```bash
npx skills add stoodiohq/agent-skills
```

Install a specific skill:

```bash
npx skills add stoodiohq/agent-skills --skill figma-diagram-design
```

## Available Skills

| Skill | Description |
|-------|-------------|
| [figma-diagram-design](./skills/figma-diagram-design/) | Apply professional color theming to Figma diagrams with automatic branch-based color inheritance |

## Compatibility

These skills work with Claude Code, Cursor, Windsurf, Codex, and other agents that support the [Agent Skills specification](https://agentskills.io).
